Ponts entre la base de données des observations et la suite bureautique OpenOffice. Ce paquet comprend des formules à utiliser dans la feuille de calcul, qui se connecteront à la base de données. Les instructions de compilation et de déploiement sont les mêmes que celles du paquet {@link org.geotools.openoffice}. Ils sont répétés (et adaptés) ici par commodités.
  1. idlc -C -cid -w -I $OO_SDK_HOME/idl net/seagis/openoffice/XObservations.idl
  2. regmerge sicade.rdb /UCR net/seagis/openoffice/XObservations.urd
  3. javamaker -BUCR -nD -Torg.constellation.openoffice.XObservations sicade.rdb $OFFICE_PROGRAM_PATH/types.rdb
  4. javap org.constellation.openoffice.XObservations
  5. javac net/seagis/openoffice/*.java
  6. jar -cfm0 sicade-SNAPSHOT.jar META-INF/MANIFEST.MF net/seagis/openoffice/*.class
  7. zip -9 sicade.uno.pkg sicade.rdb sicade-SNAPSHOT.jar META-INF/manifest.xml
  8. unopkg add sicade.uno.pkg
