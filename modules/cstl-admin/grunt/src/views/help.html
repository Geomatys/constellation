Add data

  1. Cliquer sur "Méta/données" dans le menu afin d'accéder au dashboard des données
  2. Cliquer sur Ajouter une donnée
  3. Choisir la source de la donnée
    • Fichier local : Données importées depuis l'ordinateur de l'utilisateur
    • Fichier serveur : Données importées depuis un serveur externe
    • Base de données : Donnée importées depuis une base de données

Edit data style

  1. Dans le dashboard de données, cliquer sur une donnée
  2. Sur le panneau de droite, cliquer sur "Styles associés" puis "Associer un nouveau style"
  3. Il est possible d'utiliser un style existant ou d'en créer un nouveau en définissant des règles

Create service

  1. Cliquer sur "Service web" dans le menu afin d'accéder au dashboard des services
  2. Cliquer sur "Créer un service" et choisir le type de service
    • WMS : Service de cartes
    • WFS : Service de données vectorielles
    • WCS : Service de grilles de données
    • WMTS : Service de cartes tuilées
    • CSW : Service de catalogues de données
    • SOS : Service de données capteur
    • WPS : Service de géotraitements
  3. Il suffit ensuite de remplir les informations sur le service (déscription + métadonnées du service)

Add data in service

  1. Cliquer sur "Service web" dans le menu afin d'accéder au dashboard des services
  2. Cliquer sur le titre d'un service ou sur le bouton "outils > éditer"
  3. Dans l'onglet "Données" cliquer sur "Ajouter"
  4. Sélectionner la donnée à ajouter et cliquer sur choisir

Create mapcontext

  1. Cliquer sur "Cartographie" dans le menu afin d'accéder au dashboard des contexte cartographiques
  2. Cliquer sur "Ajouter"
  3. Dans l'onglet "Information" compléter les information sur le contexte
  4. Dans l'onglet "Couches", cliquer sur "Ajouter une couche" puis sélectionner la source des données à ajouter
    • Interne : Données stockées sur un service WMS de Constellation-SDI
    • Externe : Données accessibles depuis un service externe via une URL
