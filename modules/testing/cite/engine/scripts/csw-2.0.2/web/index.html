Overview

This test suite is based on the following OGC specifications:

  • OGC Catalogue Services Specification, version 2.0.2 [OGC 07-006r1]
  • OpenGIS Web Services Common Specification, version 1.0.0 [OGC 05-008c1]
  • Filter Encoding Implementation Specification, version 1.1.0 [OGC 04-095]
  • Geographic information -- Geography Markup Language [OGC 03-105r1]

The conformance tests included here cover basic search and retrieval capabilities that all CSW-based catalogues must implement; these tests are intended to be incorporated into profile-specific test suites.

What is tested

  • GetCapabilities, GET method
  • DescribeRecord, POST method with XML content
  • GetRecordById, GET method
  • GetRecords, POST method with XML content

What is not tested

  • any optional capabilities

Test data

The CSW-2.0.2 test data are available in the common CSW record format. Loading the records into the catalogue is a precondition for compliance testing. While the test data are represented as a collection of csw:Record instances, no assumption is made about which application profile is supported by the implementation. It is expected that each profile defines a suitable mapping for csw:Record items.
